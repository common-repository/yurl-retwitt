== Installation == 

1. Unzip the archive in your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Settings / yURL" to edit your preferences, do not forget to check the box (activates the link)
4. Place <?php yurl_show(); ?> anywhere in your template files



== Description ==

This plugin will allow your readers to publish a RT (Re-Twitt) on twitter, quoting the user name of your choice, your post's (or page) title and its url (shortened by http://yurl.fr/ )
